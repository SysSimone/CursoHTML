Descrição do curso

Este curso apresenta conceitos básicos de web e mostra as partes e funções das tags em um documento HTML.
Ensina, também, a formatar uma página, inserir imagens e muito mais!
Tudo isso, com explicações simples e de fácil compreensão.

Estrututa do curso

  1. MODULO 1 - Introdução
  2. MODULO 2 - HTML e web
  3. MODULO 3 - Criando um documento HTML
  4. MODULO 4 - Formatando um documento HTML
  5. MODULO 5 - Criando listas
  6. MODULO 6 - Inserindo imagens
  7. MODULO 7 - Criando links
  8. MODULO 8 - Combinando tags
  9. MODULO 9 - Usando imagens mapeadas
